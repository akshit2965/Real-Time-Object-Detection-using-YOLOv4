# Real-Time Object Detection using YOLOv4

A real-time object detection system built using Python, OpenCV, and YOLOv4.

## Features

- Real-time object detection
- 80 COCO object classes
- Bounding boxes with confidence scores
- Webcam support
- Non-Maximum Suppression (NMS)

## Technologies

- Python
- OpenCV
- YOLOv4
- NumPy

## 📥 Download YOLOv4 Weights

The model weights are not included because GitHub limits file sizes.

Download them from the official source:

https://github.com/AlexeyAB/darknet/releases/download/darknet_yolo_v3_optimal/yolov4.weights

Place the downloaded file in the project root:

```

real-time-object-detection-yolov4/

    yolov4.weights

```
## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python object.py
```

Press **Q** to quit.

## Files

- object.py
- yolov4.cfg
- yolov4.weights
- coco.names
